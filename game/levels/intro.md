This is a **scaffold** for the ∞-Yoneda Game. It carries one starter level so the
[`rzk-game`](https://github.com/rzk-lang/rzk-game) engine and the deploy pipeline
can be exercised end to end. The real content — following Emily Riehl's
geodesic to the Yoneda lemma — will be filled in over later updates.
