A function $g : A \to B$ does more than map points: it carries whole cells. Applying $g$ along a path gives a path, so a function acts on morphisms, not just points. This is **functoriality**.

This section opens with a quick **pre-test**. If functoriality is new to you, say so: you will get a pointer to review first, and the dependent level will wait for you.

*By the end you will be able to:* carry a point and a morphism along a function.
